How can I help you today?

Ask me anything about UCM student services including admissions, financial aid, housing, transcripts, registrar, and IT support. Select a topic above to get suggested questions.

This agent uses a RAG pipeline built on scraped UCM public pages, embedded into Pinecone and retrieved at query time, with a LangGraph agent calling Groq LLaMA 3.1 to generate answers. Workflow automation runs through n8n with a human-in-the-loop review layer via Google Sheets.

Stack Python · FastAPI · LangGraph · LangChain · Groq LLaMA 3.1 · Pinecone · n8n · Render · GitHub Pages
